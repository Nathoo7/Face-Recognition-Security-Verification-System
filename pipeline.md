1. Run train.py to train "train_me" folder ("imperial_dataset" for Imperial dataset ) with training, validation and test set. create "checkpoints" folder with weight (best_model.pth). 
2. Run Infer.py to extract the embeddings from the training set.
3. Run testing.py to compare training set with test set for evaluation.
  
Ps.: datasets: https://drive.google.com/drive/folders/1XHgTCr-zgWh2E0ym7B3ZGVifB2VvmSBj?usp=drive_link
     models: https://drive.google.com/drive/folders/1Na67zuvW13qQUgwmfrFBJvN27jvdYCNZ?usp=drive_link

For facial security system, run Face_System with both models and make sure to create 2 json files for each models (face_descriptors_imperial.json and face_descriptors_casia.json)
